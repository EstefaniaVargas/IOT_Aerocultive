¿Qué es Aerocultive?
❮ ❯

Aerocultive es un sistema vertical para agricultura aeropónica en el hogar, en donde se monitorean variables físicas del cultivo como pH, intensidad lumínica, humedad y temperatura, de modo que el usuario tenga acceso a la información de las condiciones de su cultivo. Asi mismo, cuenta con un sistema de control para mantener los valores de humedad y pH en los rangos óptimos para la planta. Aerocultive brinda la posibilidad de cultivar el tipo de planta que el usuario seleccione en la aplicación móvil de entre tres tipos de planta: Lechuga, Cilantro y Ajo. Adicional a la aplicación móvil, el sistema está vinculado a la página web de modo que el usuario puede tener información en tiempo real sobre su cultivo desde cualquier locación.
Nuestros Usuarios
Aerocultive está diseñado para aquellas personas que quieran hacer parte de la agricultura urbana, aquellos que quieren tener un pequeño huerto en el hogar, aquellos que quieren experimentar nuevas técnicas de agricultura, más específicamente cualquier persona que quiera estar junto a nosotros mejorando la calidad de los vegetales que se consumen.
